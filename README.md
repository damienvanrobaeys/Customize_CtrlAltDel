# Customize CtrlAltDel buttons: Hide or display buttons

<img src="https://github.com/damienvanrobaeys/Customize_CtrlAltDel/blob/main/ctrlaltdel_preview.JPG" width="500" align="center">

**!!! To run with admin rights !!!**


**Available scripts**
Allows you to hid or display below buttons from the CtrilAltDel
- Sign out
- Lock
- Change password
- Task manager
- Switch user

**Available scripts**
- Function\Set_CtrlAltDel.ps1: Run from a function like Set_CtrlAltDel -DisableAll
- With params\Set_CtrlAltDel.ps1: Run from a script like .\Set_CtrlAltDel.ps1 -DisableAll


**Available parameters**
- DisableLockWorkstation
- DisableTaskMgr
- DisableLogOff
- DisableChangePWD
- DisableSwitchUser
- DisableAll
- EnableLockWorkstation
- EnableTaskMgr
- EnableLogOff
- EnableChangePWD
- EnableSwitchUser
- EnableAll
