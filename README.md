# Customize CtrlAltDel buttons: Sign out, Lock Task

**Available scripts**
Allows you to hid or display below buttons from the CtrilAltDel
- Sign out
- Lock
- Change password
- Task manager
- Switch user

**Available scripts**
- Function\Set_CtrlAltDel.ps1: Run from a function like Set_CtrlAltDel -DisableAll
- With params\Set_CtrlAltDel.ps1: Run from a script like .\Set_CtrlAltDel.ps1 -DisableAll


**Available parameters**
- DisableLockWorkstation
- DisableTaskMgr
- DisableLogOff
- DisableChangePWD
- DisableSwitchUser
- DisableAll
- EnableLockWorkstation
- EnableTaskMgr
- EnableLogOff
- EnableChangePWD
- EnableSwitchUser
- EnableAll
